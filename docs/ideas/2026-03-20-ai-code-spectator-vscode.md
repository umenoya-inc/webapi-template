# AI コーディング観戦用 VSCode 拡張

## 概要

ターミナルベースの AI コーディングツール（Claude Code 等）と連携し、AI の作業をリアルタイムで可視化する VSCode 拡張のアイデア。AI が読み書きしているファイルを自動的に開き、変更箇所にスクロールする「観戦モード」を提供する。

## 想定機能

- AI が Read/Edit/Grep したファイルがツリー上にハイライトされ、該当行に自動スクロール
- 変更の diff がエディタ上にインラインでライブ表示
- AI が現在コンテキストに持っているファイル群をツリービューで可視化
- ツリーやシンボルをクリックして AI に「ここを見て」と伝える双方向連携

## 背景

- バイブコーディングではターミナルのログを追うしかなく、コードベース全体での「今どこにいるか」が見えにくい
- VSCode の拡張 API（ツリービュー、デコレーション、diff 表示、ターミナル連携）が揃っているため、IDE のベース部分を作る必要がない
- Claude Code はツール実行イベントをフック可能で、Read/Edit/Grep のイベントをビューワーに流すだけで最低限の体験は実現できそう

## 既存の類似ツール

- **Claude Code Dashboard** — コミュニティ製 VSCode 拡張。セッションデータからファイル操作を可視化。最も近いが観戦モード + 双方向連携はない
- **Cursor** — IDE 内蔵のインライン diff 表示。ターミナルベースツールとの連携はない
- **AgentPrism** — エージェント実行トレースのツリー表示ライブラリ。コード可視化ではなくワークフロー可視化向け

## 備考

- このプロジェクト（Web API Template）とは直接関係のない汎用アイデア
